#Pac-Man Game

A Pac-Man game implemented in C++ using the SFML library.

#Features
- Classic maze gameplay
- Ghost AI using A* pathfinding
- Multiple difficulty levels
- Score tracking and leaderboards
- Power-ups and red danger zones

#Requirements
- C++17 compatible compiler
- SFML 3.0
- CMake 3.10+ (optional)

#Installation
- Linux / macOS
